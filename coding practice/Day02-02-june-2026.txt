## Ansible - install & Enable nginx

--- 
- name: Install and start Nginx on Pathnex
  hosts: all
  become: yes

  tasks:
  -name: install nginx
  yum:
  name: nginx
  state: present
  -name: Enable nginx
  services:
  name: nginx
  state: started
  enabled: yes
